# nevesjp.github.io

Página inicial de exemplo para GitHub Pages.

Como usar:
1. Edite `index.html` e faça commit na branch `main`.
2. Vá em _Settings_ → _Pages_ no GitHub e confirme que a fonte (source) é `main` / `root`.
3. Aguarde alguns minutos e acesse: https://nevesjp.github.io

Dicas:
- Para páginas estáticas simples, manter apenas HTML/CSS é suficiente.
- Para blog ou site com templates, considere usar Jekyll (suportado nativamente pelo GitHub Pages) ou frameworks estáticos (Hugo, Eleventy).
